# Operator Training Checklist (Store/Data Entry)

Use this checklist during training and first 2 weeks of go-live.

## A. Basic Skills
- Can log in and log out correctly.
- Can select correct date and shift.
- Understands item name and unit (L, Kg, Nos).
- Understands difference between inward and outward.

## B. Inward Entry
- Can create a new inward transaction.
- Can enter challan/invoice number.
- Can add supplier name.
- Can save entry without missing fields.

## C. Outward Entry
- Can create outward issue entry.
- Can select correct department/purpose.
- Can save outward quantity correctly.

## D. Special Entries
- Can record return material.
- Can record wastage entry with reason.
- Knows when to call supervisor for correction.

## E. Report Usage
- Can open current stock report.
- Can open daily movement report.
- Can identify low-stock items.

## F. Daily Discipline
- Enters transactions same shift (no backlog).
- Keeps physical documents properly.
- Does end-of-shift reconciliation.

## G. Practical Test (Trainer to Mark)
- Inward test done: Pass / Fail
- Outward test done: Pass / Fail
- Correction request test done: Pass / Fail
- Report check test done: Pass / Fail
