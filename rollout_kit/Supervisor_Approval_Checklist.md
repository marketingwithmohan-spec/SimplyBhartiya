# Supervisor Approval Checklist

Use this checklist for daily control and audit discipline.

## 1. Start of Day Checks
- Yesterday closing stock report reviewed.
- Pending correction requests checked.
- Low-stock critical items reviewed.

## 2. Correction Approval Rules
Before approving any correction, verify:
- Original entry details are visible.
- Physical proof exists (challan/slip/email approval).
- Quantity/unit is correct.
- Reason is clearly written.

Approve only if all above are correct.

## 3. Red-Flag Cases (Mandatory Escalation)
- High quantity change.
- High-value item correction.
- Back-dated entry beyond allowed limit.
- Repeated corrections by same user.

## 4. Shift-End Checks
- Daily inward/outward totals reviewed.
- Top 10 critical items spot-checked with physical stock.
- All correction actions closed (approved/rejected).

## 5. Weekly Review
- Number of corrections per operator.
- Stock mismatch incidents.
- Stock-out incidents.
- Aging of pending entries.
