# SOP: Daily Inventory Process (Oil Extraction Company)

## 1. Purpose
This SOP helps store and production teams keep stock records correct every day.

## 2. Who Will Use This
- Store Operator
- Shift Supervisor
- Plant/Operations Manager

## 3. Scope
Use this process for all daily movement of:
- Crude oil / finished oil
- Chemicals
- Drums / packaging material
- Spare parts / consumables

## 4. Basic Rules
- Enter every stock movement on same day.
- Do not keep pending paper slips for later entry.
- Do not change old entries without supervisor approval.
- Every correction must have reason.

## 5. Daily Workflow (Simple)

### Step 1: Start of Shift
- Open system.
- Check yesterday closing stock report.
- Confirm no pending unapproved corrections.

### Step 2: Material Inward (Stock Received)
- Select item.
- Enter quantity and unit.
- Enter supplier/challan number.
- Save entry.
- Put physical inward slip in file.

### Step 3: Material Outward (Stock Issued)
- Select item.
- Enter quantity and unit.
- Enter issue purpose/department.
- Save entry.

### Step 4: Return / Rejection / Wastage
- Create separate entry type (not normal issue).
- Add reason clearly.
- Save and inform supervisor if large quantity.

### Step 5: End of Shift
- Open daily summary report.
- Verify total inward and outward.
- Match with physical register for high-value items.
- Hand over to next shift.

## 6. Correction Process
- If wrong entry is found:
1. Operator raises correction request.
2. Supervisor checks paper proof.
3. Supervisor approves/rejects in system.
4. Reason is mandatory.

## 7. Low Stock Process
- Check low-stock report at least once per shift.
- Inform purchase/store head if any item is below minimum level.
- Mark urgent items clearly.

## 8. Escalation
- If system is down more than 15 minutes:
1. Continue entry in emergency paper format.
2. Inform supervisor and IT/contact person.
3. Update system once restored.

## 9. Reports to Review Daily
- Current stock report
- Inward/outward summary report
- Low-stock alert report

## 10. Audit Readiness
- Keep challan/inward slips arranged date-wise.
- Keep approval reasons for corrections.
- No blank fields for quantity/unit.
