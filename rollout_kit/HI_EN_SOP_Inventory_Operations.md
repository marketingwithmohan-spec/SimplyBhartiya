# SOP: Daily Inventory Process (Hindi + English)

## 1) Purpose / Uddeshya
This SOP keeps daily stock records accurate and simple.
Yeh SOP daily stock record ko sahi aur asaan banata hai.

## 2) Users / Kaun Use Karega
- Store Operator / Store operator
- Shift Supervisor / Shift supervisor
- Plant Manager / Plant manager

## 3) Scope / Lagu Kshetra
- Oil, chemicals, drums, spare parts
- Tel, chemicals, drums, spare parts

## 4) Golden Rules / Mukhya Niyam
- Enter every movement on same day.
- Har stock movement same day enter karein.
- No backdated changes without supervisor approval.
- Supervisor approval ke bina purani entry change na karein.
- Every correction must have reason.
- Har correction ke saath reason mandatory hai.

## 5) Daily Steps / Roz ka Process
1. Start shift: check yesterday closing report.
   Shift start par kal ka closing report check karein.
2. Inward: item, qty, unit, challan/supplier fill karke save karein.
3. Outward: item, qty, issue purpose fill karke save karein.
4. Return/wastage: separate entry type use karein.
5. End shift: daily summary verify karein.

## 6) Correction Process / Galat Entry Sudhar
1. Operator correction request raise kare.
2. Supervisor proof verify kare.
3. Approve/Reject with reason.

## 7) Low Stock / Kam Stock Alert
- Check low stock report once per shift.
- Har shift me low stock report check karein.
- Inform purchase for urgent items.
- Urgent item ke liye purchase team ko inform karein.

## 8) Downtime Process / System Band Hone Par
1. Emergency paper format use karo.
2. Supervisor ko batao.
3. System chalu hone par entries update karo.

## 9) Daily Reports / Roz ke Reports
- Current stock
- Inward/Outward summary
- Low stock alert
