# One-Page A4 Kit (Print and Paste Near Workstation)

## A) Operator Quick Flow
1. Shift start: check yesterday closing.
2. Inward: item + qty + unit + challan + save.
3. Outward: item + qty + purpose + save.
4. Return/Wastage: separate entry + reason.
5. Shift end: verify daily summary.

## B) Supervisor Quick Control
1. Check pending corrections.
2. Approve only with proof + reason.
3. Escalate high-value/high-qty changes.
4. End shift: review totals + spot-check stock.

## C) Daily Do and Don't
Do:
- Same-day entry
- Clear reasons for corrections
- Keep documents date-wise

Don't:
- Backdated edits without approval
- Mixed units for same item
- Pending slips for later entry

## D) Emergency Downtime Process
1. Use emergency paper format.
2. Inform supervisor.
3. Update system when restored.
